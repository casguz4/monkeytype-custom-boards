# Monkey Type User Compare

This is an application to compare typing statistics on [monkeytype.com](https://monkeytype.com) between different users.

It allows you to see how your typing speed and accuracy stack up against others.

## Features

- Compare all stateistics including WPM, accuracy, and more.
- see a clear visual representation of who is better at what.
- Simple
